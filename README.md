# Research

## projects.json

Edit this file when you start a new project or its status changes. Example of a project definition:
```json
{
  "name": "My new project",
  "directory": "00-my-new-project",
  "description": "This is my new project.",
  "status": "proposal"
}
```

Status can be any of the following:
* proposal
* active
* paused
* completed
* dropped
